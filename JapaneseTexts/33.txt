友 は 別に 追窮 し なかっ た 。 其後 暫 らく し て 自分 は 何時のまにか 鶴 は 自分 を 恋し て い て くれ た の だ が 父 や 母 や 兄 の すすめ で 進ま ず ながら 人妻 に なっ た の だ と 理由 も なし に 思う ょう に なっ た 。 そう し て それ から 一月 も たっ た 。 今 は 鶴 を あわれむ ょうな 気分 に なっ た 。 そうして 鶴 の 運命 が 気 に なり だし た 。 自分 は この 感じ が あやまっ て いる か 、 い ない か を 鶴 に 逢っ て 聞き たく 思っ て いる 。 しかし 鶴 が 「 私 は 一 度 も 貴君 の こと を 思っ た こと は あり ませ ん 」 と 自ら 云お う とも 、 自分 は それ は ロ だけ だ 、 少く も 鶴 の 意識 だけ だ と 思う に ちがい ない 。
